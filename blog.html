Question-1:What are the different ways to select an element in the DOM?

Answer:

  • There are multiple ways to select an element in the DOM. Some of the most common methods include-
  • 1.getElementsByTagName
  • 2.getElementById
  • 3.getElementsByClassName
  • 4.querySelector
  • 5.querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent?

Answer:

  • 1.innerHTML: It sets or returns the HTML content of an element.
  • 2.innerText: It sets or returns the text content of an element.
  • 3.textContent: It sets or returns the text content of an element, including all text and whitespace.

Question-3: What is event delegation in the DOM?

Answer:

  • Event delegation is a technique in which we attach a single event listener to a parent element that will fire for all descendants matching a selector, whether they are present now or added in the future.

Question-4: What is event bubbling in the DOM?

Answer:

  • Event bubbling is a mechanism in the DOM where an event starts from the target element and bubbles up to its ancestor elements (parent, grandparent, etc.).

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:

  • In JavaScript, you can dynamically create, add, and remove elements using the DOM (Document Object Model).
  • 1.Create an Element → Use document.createElement("tag").
  • 2.Add an Element → Use .appendChild(), .append(), .prepend(), or .insertBefore() to insert it into the DOM.
  • 3.Remove an Element → Use .removeChild() or .remove() to delete it from the DOM.
