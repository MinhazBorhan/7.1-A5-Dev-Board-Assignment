Question-1:What are the different ways to select an element in the DOM?

Answer:

  • There are multiple ways to select an element in the DOM. Some of the most common methods include-
  • 1.getElementsByTagName
  • 2.getElementById
  • 3.getElementsByClassName
  • 4.querySelector
  • 5.querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent?

Answer:

  • 1.innerHTML: It is used to sets or returns all the HTML content of an element includes hidden text.
  • 2.innerText: It is used to sets or returns only visible text content of a HTML element.
  • 3.textContent: It is used to sets or returns all text in a HTML element. It ignore HTML element(tag) and return only the text of a HTML element.

Question-3: What is event delegation in the DOM?

Answer:

  • Event delegation is a technique in JavaScript where multiple child elements are handled by adding an event listener to the parent element.

Question-4: What is event bubbling in the DOM?

Answer:

  • Event bubbling is a concept in the DOM where an event starts from the target child element and bubbles up to its parent elements (parent, grandparent, etc.) in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:

  • In JavaScript, you can dynamically create, add, and remove elements using the DOM (Document Object Model).
  • 1.Create an Element → document.createElement("tag").
  • 2.Add an Element → .appendChild(), .append() to insert it into the DOM.
  • 3.Remove an Element → .removeChild() or .remove() to delete it from the DOM.
